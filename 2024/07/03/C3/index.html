sizeof运算符

用于计算变量或类型的大小，结果为无符号整数类型（size_t），单位是字节。

sizeof 是一个在编译时求值的运算符，用于计算类型或变量所占的字节数。它不会实际计算表达式的值，而是只看表达式的类型。

sizeof(c = a + b) 中 c = a + b 不会被执行，sizeof 只关心 c 的类型。sizeof(c = a + b) => sizeof(short)

整形提升

在 C 语言中，当进行算术运算（如 +）时，如果操作数的类型比 int 小（如 char 或 short），会先将它们提升为 int。

字符常量：在C语言中，字符常量（例如’a’）的类型实际上是int，它的大小通常是4字节

sizeof(“a”)：在 C 语言中，字符串常量被表示为一个以空字符 \0 结尾的字符数组。”a” 是一个包含两个字符的数组：{‘a’, ‘\0’}。

字符串常量

char arr[] = “hello”; 的工作原理：
在这种情况下，字符串 “hello” 的内容被复制到数组 arr 中。
数组 arr 是一个独立的字符数组，它存储了字符串的内容，而不是直接指向字符串常量的地址。
编译器会分配一块内存给数组 arr，并将 “hello” 的字符内容逐个复制到数组中。
内存布局可能如下：
arr: {‘h’, ‘e’, ‘l’, ‘l’, ‘o’, ‘\0’}

字符串：字符串的结束标志是一个’\0’的转义字符。

字符串常量（”abc”）会在数组初始化时包括一个额外的终止符 ‘\0’。当字符数组进行部分初始化时，未显式初始化的元素会被自动初始化为’\0’，实际char arr3[5] = {‘a’, ‘b’, ‘c’, ‘\0’, ‘\0’}。

strlen函数(string length)：从字符串的起始地址开始，逐个检查字符，直到遇到第一个空字符（’\0’）为止，返回值为已计数的字符数。

const限定符：用于限定一个变量为只读，即常变量 - 具有常属性（不能被改变的属性），但本质上仍为变量。

static修饰符：修饰局部变量的时候，局部变量出了作用域不会销毁。
