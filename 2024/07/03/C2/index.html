内存

作者: Suno 时间: July 3, 2024  17:26:49 类别: C

在计算机中内存是以位为最小存储单位，把8位划分成一个内存单元，每个内存单元的大小是1字节。

为了能够有效的访问到内存的每个单元，就给内存单元进行了编号，这些编号就被称为该内存单元的地址。

数据存取

CPU < —— > 寄存器 < —— > 高速缓存 < —— > 内存 < —— > 硬盘

地址编码

32位操作系统的环境下内存地址是由8位十六进制数编址，可以表示的地址范围是从 0x00000000 到 0xFFFFFFFF，一共 2^32（16 * 16 * 16 * 16 * 16 * 16 * 16 * 16）个地址，每个地址对应 1 字节，因此 CPU 最大可寻址的内存为 4GB。

int a = 1; 整型变量a在内存中要分配4个字节的空间。

当取地址a时拿到的是第一个字节的地址
