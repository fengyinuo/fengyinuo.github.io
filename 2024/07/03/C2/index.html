内存

作者: Suno 时间: July 3, 2024  17:26:49 类别: C

如要更好的理解指针和数组，就要理解“内存”

内存是电脑上特别重要的存储器，计算机中所有程序的运行都是在内存中进行的。

所以为了有效的使用内存，就把内存划分成一个个小的内存单元，每个内存单元的大小都是一字节。

为了能够有效的访问到内存的每个单元，就给内存单元进行了编号，这些编号就被称为该内存单元的地址。

数据的存取

CPU < —— > *寄存器 < —— > 缓存 < —— > 内存 < —— > 硬盘

内存单元的地址编码

32位计算机指的是该计算机系统中的数据处理单元（如CPU寄存器、内存地址线等）能够处理的数据宽度为32位，即它可以存储或运算的最大整数是4字节（32比特）。

即对应了寄存器每次的寻址能力为32位。

32位对应了32根地址线，即真实存在的物理线。当通电之后，电信号转换为数字信息，便存储了1和0组成的二进制序列。

即地址编码为32位二进制数，同时2的32次方个地址在内存中对应有2的32次方个内存单元。

int a = 1; 整型变量a在内存中要分配4个字节的空间，每个内存单元的大小是1个字节，即需要4个内存单元存放a。

当取地址a时拿到的是第一个字节（内存单元）的地址
