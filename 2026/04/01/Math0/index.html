高数

Author: suno Date: April 1, 2026  20:06:09 Category: Math

数集

自然数集（N）：0, 1, 2, 3, 4

整数集（Z）：-2, -1, 0, 1, 2

有理数集（Q）：0.5, 0.333…, $\frac pq$; 整数、有限小数、无限循环小数（可以表示为两个整数之比的实数）

无理数集：π, e, $ \sqrt{2} $, $ \sqrt[3]{2} $; 无限不循环小数（不能写成两个整数之比的实数）

实数集（R）：有理数和无理数的总称

不等式

$a + b \ge 2\sqrt{ab}$      { a, b | a > 0, b > 0 }
$ab \leq (\frac {a+b}2)^2$      { a, b | a $\in$ R, b $\in$ R }
对于分式不等式：$\frac {2-x}{x+3}$ > 0，两边同时乘以分母的平方 $(x + 3)^2$：(2 - x)(x + 3) > 0
绝对值的几何意义：在数轴上，一个数a所对应的点与原点的距离称为该数的绝对值，记作：|a|；|a - b|表示数轴上 a 、b 两个数字之间的距离。

指数函数

$a^x = N$ ( a > 0, $a \neq 1$ )      a（底数） x（指数） N（幂）

  • 当a < 0，取a = -2、x = $\frac 12$，y = $(-2)^{\frac 12}$ = $ \sqrt{-2} $
  • 当a = 0
    • 当x < 0，取x = -1，y = $0^{-1}$ = $\frac 10$
    • 当x = 0，y = $0^0$
  • 当a = 1，y = $1^x$ = 1

{ x | $x \in R$ }      ( 0, $+\infty$ )

$a^m · a^n$ = $a^{m+n}$      $a^m / a^n$ = $a^{m-n}$

对数函数

$log_aN = b$ ( a > 0, $a \neq 0$ )      a（底数） N（真数） b（对数）

{ x | x > 0 } ( $-\infty$, $+\infty$ )

$log_ax + log_ay$ = $log_a(x+y)$      $log_ax - log_ay$ = $log_a(x/y)$

幂函数

y = $x^a$

  • y = $x^2$      { x | $x \in R$ }      [ 0, $+\infty$ )
  • y = $x^{\frac 12}$      { x | $x \ge 0$ }      [ 0, $+\infty$ )

三角函数

$a^2$ + $b^2$ = $c^2$
$sin \theta$ = ${\frac ac}$
$cos \theta$ = ${\frac bc}$
$tan \theta$ = ${\frac ab}$
$cot \theta$ = ${\frac ba}$
$\angle 30^\circ : 1:\sqrt{3}:2$
$\angle 45^\circ : {\frac 1{\sqrt{2}}}:{\frac 1{\sqrt{2}}}:1$

二次函数

y = $ax^2$ + bx + c

$ax^2$ + bx + c = 0

  • 当Δ = $b^2$ - 4ac $\ge$ 0时, x = $\frac {-b \pm \sqrt{4ac - b^2}}{2a}$
    • $x_1 + x_2$ = $- \frac ba$
    • $x_1 * x_2$ = $\frac ca$
  • 当Δ = $b^2$ - 4ac < 0时, 无实数根

对称轴：$x = - \frac b{2a}$

$a^2$ - $b^2$ = (a + b)(a - b)
$(a \pm b)^2$ = $a^2$ $\pm$ 2ab + $b^2$

$ax^2$ + bx + c = ($a_1x$ + $c_1$)($a_2x$ + $c_2$)

  • a = $a_1$ * $a_2$
  • c = $c_1$ * $c_2$
  • b = $a_1c_2$ + $a_2c_1$
