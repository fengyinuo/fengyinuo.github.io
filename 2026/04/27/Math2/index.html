概率

Author: suno Date: April 27, 2026  12:06:05 Category: Math

阶乘

n! = n(n-1)(n-2)···3 * 2 * 1 (0! = 1)

排列

$A^m_n$ = n(n-1)(n-2)···(n-m+1) = $\frac {n!}{(n-m)!}$ ($A^0_n$ = 1)

组合

$C^m_n$ = $\frac {A^m_n}{A^m_m}$ = $\frac {n!}{m!(n-m)!}$

$C^m_n$ = $C^{n-m}_n$ ($C^0_n$ = 1)

集合

$\emptyset$ 为任意集合的子集      $\emptyset$ 为任意非空集合的真子集

$\emptyset \in$ { $\emptyset$ }      $\emptyset \subseteq$ { $\emptyset$ }      $\emptyset \subsetneqq$ { $\emptyset$ }

A={$\emptyset$, 1, 2}      $\emptyset \in A，\emptyset \subseteq A，\emptyset \subsetneqq A$
A={ {$\emptyset$}, 1, 2}      $\emptyset \notin A，${$\emptyset$} $\in A，\emptyset \subseteq A，\emptyset \subsetneqq A$

A={1, 2}，B={1, 2}，C={1, 2, 3}      $A \subseteq B，B \subseteq A，A(/B) \subseteq C，A(/B) \subsetneqq C$

A={1, 2, 3, …, n}      A 的子集个数为 $2^n$（包含空集）

随机事件

我们将随机试验 $E$ 所有可能结果组成的集合称为 $E$ 的样本空间，记作 $\Omega$。$\Omega$ 的元素（即 $E$ 的每个结果）称为样本点，样本点一般用 $\omega$ 表示。一般地，称实验 $E$ 的样本空间 $\Omega$ 的子集为 $E$ 的随机事件，简称事件，常用大写字母 $A，B，C，···$ 表示，它是样本空间 $\Omega$ 的子集合。

运算法则

$A \cup B = A + B，A \cap B = AB，\Omega - A = \overline{A}$

交换律：$A \cup B = B \cup A，A \cap B = B \cap A$

结合律：$(A \cup B) \cup C = A \cup (B \cup C)，(A \cap B) \cap C = A \cap (B \cap C)$

分配律：$(A \cup B) \cap C = (A \cap C) \cup (B \cap C)，(A \cap B) \cup C = (A \cup C)(B \cup C)$

德摩根律：$\overline{A \cup B} = \overline{A} \cap \overline{B}，\overline{A \cap B} = \overline{A} \cup \overline{B}$

差化积：$A - B = A \cap \overline{B}$

吸收律：$A \cup \Omega = \Omega，A \cap \Omega = A，A \cup \varnothing = A，A \cap \varnothing = \varnothing$

事件表示

A、B、C都不发生：$\overline{A}\overline{B}\overline{C}$

A发生，B、C不发生：$A \overline{B} \overline{C}$

A、B发生，C不发生：$AB\overline{C}$

A、B、C恰好一个发生：$A\overline{B}\overline{C} \cup \overline{A}B\overline{C} \cup \overline{A}\overline{B}C$

A、B、C恰好两个发生：$AB\overline{C} A\cup \overline{B}\overline{C} \cup \overline{A}BC$

A、B、C至少一个发生：$A \cup B \cup C$

A、B、C至少两个发生：$AB \cup BC \cup AC$

A、B、C至多一个发生：$\overline{AB \cup AC \cup BC}$

A、B、C至多两个发生：$\overline{ABC}$

概率公式

对任意两个事件 $A，B$ ：

P(B - A) = P(B) - P(AB)

P(A $\cup$ B) = P(A) + P(B) - P(AB)

P(B | A) = $\dfrac{P(AB)}{P(A)}$      P(AB) = P(A)P(B | A) = P(B)P(A | B)

全概率公式与贝叶斯公式

  • $A_1，A_2，A_3，···，A_n$ ( P($A_i$) > 0 ，$A_{i-1}A_i$ = $\varnothing$)
  • $A_1 \cup A_2 \cup A_3 \cup ··· \cup A_n = \Omega$

$P(A) = \sum_{i=1}^{n} P(B_i)P(A\mid B_i)$

$P(B_i \mid A) = \dfrac{P(B_iA)}{P(A)} = \dfrac{P(B_i)P(A\mid B_i)}{\displaystyle\sum_{i=1}^{n} P(B_i)P(A\mid B_i)}$
