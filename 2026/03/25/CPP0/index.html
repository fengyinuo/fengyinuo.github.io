算术转换

在 C 语言中，当进行比较运算（如 <）时，如果操作数的类型不同，其中一个是无符号类型，另一个是有符号类型，会将有符号类型转换为无符号类型。
这里，a 的值是 -1，但在与 b（无符号整型）比较时，a 会被转换为无符号整型，即4294967295。

整形提升

在 C 语言中，当进行算术运算（如 +）时，如果操作数的类型比 int 小（如 char 或 short），会先将它们提升为 int。

sizeof运算符

用于计算变量或类型的大小，结果为无符号整数类型（size_t），单位是字节。

sizeof 是一个在编译时求值的运算符，用于计算类型或变量所占的字节数。它不会实际计算表达式的值，而是只看表达式的类型。

sizeof(c = a + b) 中 c = a + b 不会被执行，sizeof 只关心 c 的类型。sizeof(c = a + b) => sizeof(short)
